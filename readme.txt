Скрипт для подключения к базе данных mysql и выполнения каких либо операций с ней.

Если проект скачан не с папкой node_modules, запустите терминал и запустите команду npm install.

Для работой с командной строкой была использована библиотека yargs.
Чтобы посмотреть все команды: node app --help.
Чтобы передать парметры в команду: node app 2 --"FIO"="FIO" --"dateBorn"="2023-05-09" --"sex"="m" (m/w)
